# GRUPO PARIA Landing

Landing page estatica para GRUPO PARIA con identidad corporativa navy/blanco, secciones de software, IPTV, infraestructura digital, producto NewTV y contacto.

## Abrir localmente

Abre `index.html` directamente en el navegador o sirve la carpeta con:

```sh
python3 -m http.server 4173
```
